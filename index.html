Whoa! Jekyll, Foundation 4, and github pages!

Simply combining my foundation 4 starter files and Jekyll to be used for github pages! See Jekyll's documentation, the foundation documentations, and GitHub pages help pages.

This is amazing. You can see all my sources here. I created this for my personal use, but if you'd like to use it, please feel free. Shout out daigo.org or @DaigoFuji are appreciated, but not required.

You should probably customize the foundation css and stick it to the css folder here, if you'd like to change colors etc. All the overrides are happening in styles.css.
